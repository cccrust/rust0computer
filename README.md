# rust0 -- 用 rust 重建簡易電腦工業

* cpu -- RISCV 處理器
* ir -- LLVM IR 中間碼
* compiler -- 簡化後的 RUST 語言 RUST0 之編譯器
* js0 -- 簡化後的 JavaScript 語言 js0 之編譯器 
* os -- 用 rust 寫的 RISCV 處理器上之作業系統
* net -- 網路相關
    * telnet -- 重新實作 telnet
    * browser -- 簡易瀏覽器
    * js0 -- JavaScript 簡化版 js0 語言解譯器
* eda -- 電子設計開發工具
    * verilog0 -- 簡化的 verilog 語言之模擬器
* ai -- 人工智慧實作
    * nn -- 神經網路套件，類似 rust candle.
    * llm -- 語言模型，類似 GPT
    * agent -- 代理人，類似 OpenClaw
